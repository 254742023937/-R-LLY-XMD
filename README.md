## ✨ Features

- Built with **Node.js**
- Uses [**Baileys**](https://github.com/whiskeysockets/Baileys) WhatsApp Web API
- AI-based chat via **OpenAI**
- Group management tools
- Media utilities
- API integration (subject to uptime availability)
